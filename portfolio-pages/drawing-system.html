Meghan Rennie

Generative Art Drawing System

  • Medium: Website
  • Software used: Processing
  • Project type: Assignment
  • Year: 2021

Meghan created this Processing sketch in response to a project prompt. The prompt asked for a drawing system in which the viewer could interact with the sketch to create different images. Instead of coding different viewer-controlled brushes to allow "traditional drawing" within the sketch, Meghan coded a few malleable bezier curves that are influenced by the mouse's position. These funky bending lines respond to mouse movements, but still conform to some of their own parameters (e.g., the coordinates of their anchor points, etc.). This results in the creation of abstract art images that are half created by the viewer's own intentions and choices, and half created by the code's response to the user's actions. The viewer can slowly learn to understand the code's behaviour as they play within the sketch, but there is still some surprise and whimsy inherent to the process and its end result.
